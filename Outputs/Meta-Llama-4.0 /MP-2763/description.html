Test Case: Verify Initiator Widget allows drag and drop of one or more PDF files

Test Case ID

TC-001

Test Scenario

Verify that the Initiator Widget allows drag and drop of one or more PDF files.

Test Entry Requirement / Prerequisites

  • Mojo for Suppliers Web App is installed and configured.
  • Initiator Widget is enabled and visible on the dashboard.
  • Test user has access to the dashboard.
  • PDF files are available for testing.

Test Tools

  • Browser console for debugging.

Test Conditions

  • Browser window is maximized.
  • Network connection is stable.

Test Environment

  • Mojo for Suppliers Web App environment.
  • Test dashboard with Initiator Widget.

Code Deployment Status

  • Latest code changes are deployed.

Test Data Validation

  • Verify that the Initiator Widget allows drag and drop of one or more PDF files.
  • Check that the files are uploaded successfully.

Test Data

  • One or more PDF files.

Browser / Device

  • Chrome v120.0.6099.129
  • Edge v120.0.2210.91
  • Firefox v121.0

Test Steps

  1. Log in to the Mojo for Suppliers Web App.
  2. Navigate to the dashboard with the Initiator Widget.
  3. Drag and drop one or more PDF files onto the Initiator Widget.
  4. Verify that the files are uploaded successfully.
  5. Check that the files are listed in the Initiator Widget.

Expected Outcome

  • The Initiator Widget allows drag and drop of one or more PDF files.
  • The files are uploaded successfully.
  • The files are listed in the Initiator Widget.

Expected Results

  • The Initiator Widget uploads the PDF files successfully.
  • The files are listed in the Initiator Widget with their file names and extensions.

Risk Coverage

  • File upload functionality.
  • Drag and drop functionality.
