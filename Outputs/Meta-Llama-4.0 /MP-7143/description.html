Test Case: TC-001 - Verify emails are sent to current managers when a call is raised with a subject that has a manager type to notify

Test Case ID

TC-001

Test Scenario

Verify that emails are sent to current managers when a call is raised with a subject that has a manager type to notify, and there are managers attached to that manager type with a status of both 'Current' and 'Archived'.

Test Entry Requirement / Prerequisites

  • Mojo v7.0.0
  • Call notification feature enabled and configured
  • Manager types and notification settings configured
  • Test user with manager role and 'Current' status
  • Test user with manager role and 'Archived' status

Test Tools

  • Email client (e.g., Outlook)

Test Conditions

  • Test environment with Mojo v7.0.0 deployed
  • Email server configured and functional

Test Environment

  • Mojo v7.0.0 test environment
  • Email server

Code Deployment Status

  • Mojo v7.0.0 code deployed to test environment

Test Data Validation

  • Verify email sent to current manager's email address
  • Verify email not sent to archived manager's email address

Test Data

  • Manager Type: Test Manager Type
  • Current Manager: John Doe (johndoe@example.com)
  • Archived Manager: Jane Smith (janesmith@example.com)
  • Call Subject: Test Call Subject

Browser / Device

  • Chrome v120.0.6099.129
  • Edge v120.0.2210.91
  • Firefox v121.0

Test Steps

  1. Log into Mojo as administrator
  2. Create a new call with the test call subject
  3. Assign the test manager type to the call subject
  4. Save and raise the call
  5. Verify emails sent to current manager's email address
  6. Verify no emails sent to archived manager's email address

Expected Outcome

  • Email sent to current manager's email address
  • No email sent to archived manager's email address

Expected Results

  • Email subject: Notification for Test Call Subject
  • Email body: contains call details and notification message

Risk Coverage

  • Ensures current managers receive notifications for calls
  • Ensures archived managers do not receive notifications for calls
